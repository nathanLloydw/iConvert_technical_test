# stack

* PHP
* Laravel
* Vue Js
* Tailwind CSS
* laravel jetstream interia 

# set up

   ## 1
      composer install
   ## 2
      npm install
   ## 3
      npm run prod
   ## 4 (IF DB CREDENTIALS ARE POPULATED IN THE DOT ENV)
      php artisan migrate:fresh --seed 
   ## 5
      php artisan serve
   ## 6
      php artisan test
